# RBG v0.9.0 Release Plan

> 目标发布时间：2026 Q4
> 上一版本：v0.8.0（当前开发中，包含 Warmup Phase 1、InPlace Scheduling 等）

## 版本定位

v0.8.0 完成了**单角色能力增强**（Warmup、InPlace Scheduling、Port Allocation）。

v0.9.0 聚焦 **多角色协作与服务发现**——让 PD 分离、E/P/D 多模态推理在生产环境真正可用。

## 特性矩阵

### Tier 1: 必须完成（Release Blocker）

| # | 特性 | KEP | 说明 | 依赖 |
|:---:|:---|:---:|:---|:---|
| 1 | **拓扑感知调度** | [KEP](../keps/topology-aware-scheduling/) | intra-role + inter-role Pod Affinity，解决 GPU 互联域和跨角色 zone 亲和 | 无（KEP + PoC 已完成） |
| 2 | **端口模板化服务发现** | [KEP](../keps/port-template-discovery/) | Role 级 base port + 顺序递增，ConfigMap portTemplates，解决 hostNetwork 跨角色端口发现 | KEP-133, KEP-171（已完成） |
| 3 | **滚动更新 Pod 复用** | 待 KEP | 语义等价检测：revision hash 不同但 spec 实质相同的 Pod 不删除重建 | 无 |
